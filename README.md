# Technologies Used:
### Foundation
- Swift5
- UIKit
- Firebase
- Firestore Cloud(Stores chats)
- UITableView
  
### Dependencies 
- [IQKeyboardManager](https://github.com/hackiftekhar/IQKeyboardManager)(A swift library allows to prevent issues of keyboard sliding up and cover UITextField/UITextView. )
